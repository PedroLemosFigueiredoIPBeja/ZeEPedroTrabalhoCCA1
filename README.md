# Projeto de Criptografia

Uma aplicação GUI para cifras clássicas e modernas.

## Cifras Implementadas

### Clássicas:
- **Vigenère**: Cifra polialfabética
- **PlayFair**: Cifra de substituição bigrâmica

### Modernas:
- **DES**: Data Encryption Standard
- **AES**: Advanced Encryption Standard
